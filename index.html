how to use

Up in
thirty seconds.

01 —
download & open
Grab the installer below. No admin rights, no setup wizard. Opens immediately.
02 —
add a task
Type in the top field, pick a priority, press Enter. List sorts automatically.
03 —
edit inline
Click any task title — edit in place, Enter to save, Esc to cancel.
04 —
adjust priorities
Toggle Customize Priority to show per-task dropdowns. List re-sorts instantly.
05 —
complete & clear
Checkbox to finish. Clear Completed wipes all done tasks at once.

about

We got tired of
bloated software.

So we built the tool we wanted. A weekend project by a small team of developers who believe software should do its job and get out of the way.

"The best interface is the one you forget you're using."

Our guiding principle.

philosophy

Software that
respects your machine.

Modern productivity apps run background services, send telemetry, and use hundreds of megabytes of RAM to show a list of text. We were frustrated, so we fixed it.

No heavyweight runtime. Written in Python with tkinter — a canvas-based task list for smooth scrolling with zero overhead.

We track nothing. No telemetry, no analytics, no crash reporter. Your tasks are stored in a plain JSON file on your machine.

Free because it should be. Task management is a basic workflow. You shouldn't need a subscription for a list of things to do.
